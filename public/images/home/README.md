## 이미지 규격 가이드

### 수평 스크롤 카드 이미지
- 권장 크기: 640x640px
- 형식: JPG 또는 PNG
- 최적화 권장

### SNS 마케팅 섹션 이미지
- 권장 크기: 다양 (카드 크기에 맞게)
- 형식: JPG 또는 PNG
- 최적화 권장
